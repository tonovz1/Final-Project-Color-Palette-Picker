# Color Palette Picker for Digital Art

A Python-based color palette picker that lets users generate, explore, and export color palettes for use in digital art and media projects. It helps solve the common challenge of choosing visually pleasing color schemes by offering palette-generation tools for digital artists.

## Features

- Generate color palettes from a base color
  - Uses color theory rules (complementary, analogous, triadic, etc.) to compute harmonious palettes from a user-selected hex color.
- Display palettes visually in a GUI window
  - Uses tkinter to render colored swatches so users can preview the full palette at a glance.
- Copy individual color values to clipboard
  - Click a swatch and copy its hex/RGB value, making it easy to bring colors into other tools.
- Save and export palettes to a file
  - Write the palette's color values to a .txt or .json file so users can reference them in other projects.
- Random palette generator
  - Generate a completely random aesthetically pleasing palette using HSL/HSV color space constraints to avoid muddy or clashing colors.

## Requirements

- Python 3.6+
- tkinter (included with Python)

## How to Use

1. Enter a base color in hex format (e.g., #FF5733)
2. Select a harmony rule from the dropdown:
   - Complementary: opposite color on the wheel (180°)
   - Analogous: adjacent colors (±30°)
   - Triadic: three equally spaced colors (120°)
   - Tetradic: four colors in a square (90°)
   - Split Complementary: two colors ±30° from the complement
3. Click "Generate" to create the palette
4. Click any color swatch to copy it to clipboard
5. Use the Export buttons to save as JSON or TXT file

## Technical Details

The application uses Python's colorsys module to convert between color spaces (RGB, HSL). Palettes are generated by:
1. Converting base color from hex to HSL
2. Applying harmony rule by calculating new hues
3. Converting results back to hex/RGB for display

Individual colors can be clicked to copy hex values to the system clipboard. Palettes can be exported to JSON (with metadata) or formatted text files.
